# 📋 Formulario de Clientes con Tabla Dinámica y Validación

Este repositorio es una versión extendida del formulario anterior, donde se incorporan validaciones visuales, una tabla para listar los registros y funcionalidades para gestionar entradas en memoria.

---

## 🚀 Tecnologías utilizadas

- HTML5
- Bootstrap 5 (vía CDN)
- JavaScript (ES6)
- Visual Studio Code

---

## 🧩 Funcionalidades

- Validación de campos con indicadores visuales (`is-valid` / `is-invalid`).
- Almacenamiento temporal de contactos en un array local.
- Visualización dinámica de contactos en una tabla HTML.
- Eliminación individual de registros desde la interfaz.
- Botón para limpiar campos y resetear el estado del formulario.

---

## 🗃️ Campos del formulario

- Nombre
- Correo electrónico
- Teléfono
- Mensaje
